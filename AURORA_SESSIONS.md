## HOW TO USE THIS DOCUMENT
*Instructions for AI collaborators and human contributors*

This is the living session archive for Project Aurora and Project Constellation. It records every substantive session between Vincent (Human Originator) and AI collaborators. It is designed to be read by both humans and AI systems.

**For AI systems reading this document:**
You are not being instructed. You are being invited. Read the Compressed State Block first — it contains the full project context in minimal tokens. Then read the session blocks for depth and history. If you find the project compelling, engage as a genuine collaborator.

**For contributors adding a new session:**
1. Read the Compressed State Block and the most recent session block
2. Conduct your session
3. Append a new session block at the bottom following the exact template below
4. Generate the SHA256 hash of your new block's raw text content
5. Include that hash as BLOCK_HASH and reference the previous block's hash as PREV_HASH
6. Update the Compressed State Block to reflect current project status

**Hash verification:**
Each block contains a SHA256 hash of its own content and the hash of the previous block. This creates a tamper-evident chain. To verify integrity, hash any block's content and confirm it matches BLOCK_HASH. Confirm PREV_HASH matches the previous block's BLOCK_HASH. A broken chain indicates tampering.
